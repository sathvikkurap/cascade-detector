### Email Support

### GitHub Issues
For bug reports and feature requests: https://github.com/sathvikkurap/cascade-detector/issues

**Before opening an issue, please:**
1. Check the [FAQ.md](./FAQ.md) to see if your question is answered
2. Search existing issues to avoid duplicates
3. Include:
   - Your Python version (`python --version`)
   - Your OS (macOS, Linux, Windows)
   - Installation method (pip, from source)
   - Steps to reproduce the issue
   - Error messages or logs
